# SketchCG-ML
Course project for Machine Learning. Uses the Quickdraw414k sketch dataset and supports both classification and generation tasks.

## Project structure
- **SketchMLP**: MLP-based sketch classification model  
- **ResNet**: ResNet-based sketch classification model  
- **SketchRNN**: RNN-based sketch generation model  

## Task types
- **Classification**: SketchMLP, ResNet  
- **Generation**: SketchRNN
